Nested JSON String Expander
Paste your text on the left. This tool will parse the outer JSON and recursively expand values that are actually JSON strings, such as "{\"name\":\"test\"}".

Input

Supports cases like outer JSON objects whose field values are JSON-encoded strings. Also tries to handle a top-level quoted JSON string.

Output


        

          If parsing fails, the message will tell you whether the text is incomplete or not valid JSON.
